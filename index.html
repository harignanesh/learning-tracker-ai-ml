Real-World Projects

Production-grade builds. React Frontend + Python Backend + AI APIs.

1. AI Creative Studio

React + FastAPI

Goal: Build a UI where users generate images and convert text to speech.

  • Frontend: React + TailwindCSS.
  • Backend: FastAPI server calling Hugging Face Free Inference API.
  • AI Tools: Stable Diffusion (Images) + Whisper/TTS (Voice).

2. AI Code Reviewer

React + Gemini

Goal: Paste code, get a strict JSON payload back highlighting bugs and fixes.

  • Focus: Structured Output Prompting. Forcing the AI to reply ONLY in JSON so React can map the data to UI components.

3. The PDF Brain (RAG)

LangChain + Chroma

Goal: Upload a PDF and chat with it. System must show citations.

  • Architecture: Python chunks the PDF -> Embeds via HuggingFace -> Stores in local ChromaDB -> LangChain orchestrates the retrieval.

4. Autonomous Web Researcher

LangGraph Agent

Goal: Ask a question. Agent searches live internet, reads sites, and writes a report.

  • Tools: Tavily Search API (Free).
  • Flow: LangGraph state machine ensures the agent loops until it finds enough high-quality data before generating the final report.
